Help commands can exist at any level: the top level (`gh`), the second level (`gh [command]`), and the third level (`gh [command] [subcommand]`). Each can be accessed using the `--help` flag, or using `gh help [command]`.

Each help page has sections:

### Required sections
- Usage
- Core commands
- Flags
- Learn more
- Inherited flags

### Other available sections
- Additional commands
- Examples
- Arguments
- Feedback
