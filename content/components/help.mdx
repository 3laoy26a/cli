There are generally three levels of help: the top level (`gh`), the second level (`gh [command]`), and the third level (`gh [command] [subcommand]`). Each help page has sections.

### Required sections
- Usage
- Core commands
- Flags
- Learn more
- Inherited flags

### Other available sections
- Additional commands
- Examples
- Arguments
- Feedback
